# Streamlit Demo

This folder contains a demo of the implemented pipeline.  


##  ⚡️ How to run the demo locally

1. **Go to the demo folder**  
   ```bash
   cd rpcm-demo/atlas
    ```
2. **Up the containers**
    ```
    docker-compose up --build
    ```

The Atlas credentials are:

- Username: admin
- Password: admin
